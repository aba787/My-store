Diagnosis

Relevant log lines: the Vite build failed while loading the config: line 72 — the build starts, line 73 — failed to load config, and the actual error is reported on line 75: "Error: PORT environment variable is required but was not provided."
This indicates vite.config.ts (loaded from artifacts/ada-platform/vite.config.ts) is throwing when process.env.PORT is not set during the Netlify build. See the file in the repo: https://github.com/aba787/My-store/blob/main/artifacts/ada-platform/vite.config.ts
Cause: the Vite configuration requires a PORT environment variable at config-load time. Netlify build runs in a CI environment where PORT is not set by default, so loading vite.config.ts throws and the build exits with code 1.

Solution

You have two straightforward ways to fix this. Pick one or both.

Option A — (recommended quick fix) Set PORT in Netlify build environment

In the Netlify UI for your site, go to Site settings → Build & deploy → Environment → Environment variables.
Add an environment variable named PORT with a sensible value (e.g. 5173 or 3000):
Key: PORT
Value: 5173
Re-run the build.
Alternatively you can set the env var in netlify.toml in your repo (commit this file):

# netlify.toml
[build]
  command = "pnpm --filter @workspace/ada-platform... run build"
  publish = "artifacts/ada-platform/dist"

[build.environment]
  PORT = "5173"
Option B — Make vite.config.ts tolerant of missing PORT (more robust)

Edit artifacts/ada-platform/vite.config.ts so it does not throw if PORT is missing, and provides a default or only requires PORT for local dev. For example:
// typescript
// at top of vite.config.ts
const port = process.env.PORT ?? '5173'; // default to 5173 if not provided

// later, where you currently use PORT (or check it), replace strict check with:
if (!port) {
  // should never happen because of the fallback, but you can log a warning instead of throwing
  console.warn('PORT not set; using default 5173');
}
Commit and push the change, then re-run the Netlify build.
